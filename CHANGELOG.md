# Changelog

Все заметные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено
- **Модульная система навигации (NavUtils):**
  - `js/navigation.js` - утилиты для работы с навигацией
  - API для создания ссылок, кнопок, карточек навигации
  - Автоматическое извлечение данных из `config.json`
  - DRY принцип - единственный источник правды
- **Динамические маршруты:** Роутер загружает карту сайта из `data/config.json`
- **Динамическая навигация:** Header меню генерируется из `config.json`
- **Централизованное управление:** Единый источник правды для всех ссылок
- **Новые страницы согласно DEVELOPMENT_PLAN.md:**
  - `pages/quantum-basics.html` - Этап 1: Квантовая механика
  - `pages/vacuum-physics.html` - Этап 2: Квантовый вакуум
  - `pages/casimir-effect.html` - Этап 3: Эффект Казимира
  - `pages/gravity-theory.html` - Этап 5: Революционная теория гравитации
  - `pages/cvl-device.html` - Этап 7: Когерентный вакуумный левитатор
- **Документация:**
  - `docs/NAVUTILS_GUIDE.md` - руководство по модульной навигации (50+ примеров)
  - `docs/DYNAMIC_ROUTES.md` - руководство по динамическим маршрутам
- **Правила проекта:**
  - Добавлены правила DRY, модульности и унификации в `.cursorrules`
  - Запрет на дублирование кода
  - Обязательное использование NavUtils для навигации

### Изменено
- `index.html`: 
  - Полный рефакторинг инициализации роутера
  - Удалены жестко закодированные маршруты и меню
  - Интеграция с NavUtils
  - Динамическая генерация навигации
- `data/config.json`: 
  - Обновлено описание проекта
  - Добавлены все новые маршруты с метаданными
  - Поля `order`, `inMenu`, `meta` для управления навигацией
- `pages/home.html`: Новое видение проекта с этапами обучения
- `pages/about.html`: Философия проекта и этапная структура развития
- `.cursorrules`: Добавлены правила DRY, модульности и унификации
- `docs/README.md`: Добавлены ссылки на новые руководства

### Удалено
- Жестко закодированные маршруты в `index.html`
- Fallback маршруты (заменены на понятное сообщение об ошибке)
- Дублирование кода навигации

### Планируется
- Добавление интерактивных визуализаций квантовых вычислений
- Раздел с новостями и статьями
- Темная тема
- Многоязычная поддержка
- Автогенерация навигационного меню из `config.json`
- Автогенерация `sitemap.xml` из `config.json`
