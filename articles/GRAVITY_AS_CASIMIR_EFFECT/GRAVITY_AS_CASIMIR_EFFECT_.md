## 1. Основной постулат

### 1.1. Формулировка

**Гравитация есть макроскопическое проявление эффекта Казимира в неоднородном вакууме.**

Масса изменяет граничные условия для квантовых полей вакуума аналогично металлическим пластинам в классическом эффекте Казимира. Разность давлений вакуума создаёт наблюдаемую гравитационную силу.

### 1.2. Ключевые утверждения

1. **Вакуум неоднороден** вокруг массы
2. **Масса = граничное условие** для вакуумных мод
3. **Градиент вакуумной энергии** создаёт силу
4. **Гравитационная постоянная** связана с плотностью вакуумных мод

## 2. Фундаментальные принципы

### 2.1. Вакуум как физическая среда

Квантовый вакуум содержит флуктуации всех полей:

```
E_vac = ∑ₖ (ℏωₖ/2)
```

где сумма идёт по всем модам k электромагнитного и других полей.

**Плотность энергии вакуума:**

```
ρ_vac = (1/V) ∑ₖ (ℏωₖ/2)
```

В свободном пространстве эта сумма расходится, но физически важны **градиенты** ρ_vac.

### 2.2. Масса как модификатор вакуума

Масса M создаёт **эффективные граничные условия** для квантовых полей.

**Механизм:** Частицы материи взаимодействуют с виртуальными частицами вакуума, изменяя спектр мод:

```
ωₖ(r) = ωₖ,₀ × √(1 + Φ(r)/c²)
```

где Φ(r) — гравитационный потенциал (который мы выведем).

### 2.3. Эффект Казимира как базовое явление

**Классический эффект Казимира** между пластинами:

```
F_Casimir = -π²ℏcA/(240d⁴)
```

**Физический смысл:** Между пластинами меньше вакуумных мод → меньше давление → пластины притягиваются.

**Обобщение на массы:** Вокруг массы M изменяется спектр мод → градиент давления вакуума → гравитация.

## 3. Механизм возникновения гравитации

### 3.1. Изменение спектра вакуумных мод вокруг массы

Рассмотрим сферическую массу M радиуса R.

**Внутри массы** (r < R):

Вакуумные моды с λ > R **подавлены** (не помещаются), как между пластинами Казимира.

Эффективная плотность вакуумной энергии:

```
ρ_vac,in = ρ_vac,0 × (1 - α·R/λ)
```

где α ~ 1 — численный коэффициент, ρ_vac,0 — плотность в свободном пространстве.

**Снаружи массы** (r > R):

Спектр мод **искажён** присутствием массы. На расстоянии r от центра:

```
ρ_vac(r) = ρ_vac,0 × (1 - β·GM/(c²r))
```

где β — константа, которую мы определим.

### 3.2. Градиент давления вакуума

Давление вакуума связано с плотностью энергии:

```
P_vac = -ρ_vac·c²/3
```

(отрицательное давление — характерная черта вакуума)

**Градиент давления:**

```
∇P_vac = -(c²/3)·∇ρ_vac
```

Подставляя ρ_vac(r):

```
∇P_vac = -(c²/3)·ρ_vac,0·β·(GM/c²)·∇(1/r)
```

```
∇P_vac = (β·ρ_vac,0·GM/3)·(r̂/r²)
```

### 3.3. Сила на пробное тело

Пробное тело массы m испытывает силу от градиента давления вакуума:

```
F = -V_body·∇P_vac
```

где V_body — объём тела.

Для тела плотности ρ:

```
m = ρ·V_body
```

```
V_body = m/ρ
```

Подставляя:

```
F = -(m/ρ)·(β·ρ_vac,0·GM/3)·(r̂/r²)
```

```
F = -(β·ρ_vac,0·m·GM)/(3ρ)·(r̂/r²)
```

### 3.4. Отождествление с гравитационной силой

Сравниваем с законом Ньютона:

```
F_Newton = -GMm/r²·r̂
```

Должно выполняться:

```
(β·ρ_vac,0)/(3ρ) = 1

β·ρ_vac,0 = 3ρ
```

**Физический смысл:** Плотность вакуумной энергии **связана** с плотностью материи!

### 3.5. Связь с гравитационной постоянной

Из условия β·ρ_vac,0 = 3ρ и размерностей:

```
ρ_vac,0 ~ ℏc/l⁴

где l — характерный масштаб (обрезание)
```

Для материи:

```
ρ ~ M/R³ ~ (E/c²)/R³
```

Приравнивая:

```
β·ℏc/l⁴ = 3·E/(c²R³)
```

```
β·ℏc⁵·R³/l⁴ = 3E
```

Гравитационная постоянная:

```
G ~ ℏc/M² ~ ℏc/(E/c²)²
```
```
G ~ ℏc⁵/E²
```

Подставляя в условие:

```
β/l⁴ ~ 1/(Gc²)
```
```
l ~ (βGc²)^(1/4)
```

При β ~ 1:

```
l ~ (ℏG/c³)^(1/2)

где:
ℏ = 1.05×10⁻³⁴ Дж·с
G = 6.67×10⁻¹¹ м³/(кг·с²)
c = 3×10⁸ м/с

l ~ √(1.05×10⁻³⁴ × 6.67×10⁻¹¹ / (3×10⁸)³)
l ~ √(7×10⁻⁴⁶ / 2.7×10²⁵)
l ~ √(2.6×10⁻⁷¹)
l ~ 5×10⁻³⁶ м
```

**НО:** Это планковская длина! Для макроскопического обрезания нужен другой масштаб.

**Альтернативная оценка** через плотность вакуума:

```
l ~ (ℏc/ρ_vac,0)^(1/4)

При ρ_vac,0 ~ 3ρ_matter ~ 3×10⁻²⁷ кг/м³:

l ~ [(1.05×10⁻³⁴ × 3×10⁸)/(3×10⁻²⁷)]^(1/4)
l ~ (10⁻¹⁷/3×10⁻²⁷)^(1/4)
l ~ (3×10⁹)^(1/4)
l ~ 70 м
```

**Характерный масштаб ~ 10-100 метров** (зависит от выбора ρ_vac,0).

Для согласования с экспериментами на масштабе мм нужна более сложная модель обрезания.

## 4. Математическая формализация

### 4.1. Модифицированная плотность вакуумной энергии

**Общая формула:**

```
ρ_vac(r, M) = ρ_vac,0·[1 - f(GM/(c²r))]
```

где f(x) — функция, удовлетворяющая:
- f(0) = 0 (вдали от массы ρ_vac → ρ_vac,0)
- f'(0) ≠ 0 (первая производная даёт гравитацию)
- f(x) ≪ 1 для слабых полей (GM/(c²r) ≪ 1)

**Простейший выбор:**

```
f(x) = x = GM/(c²r)
```

Тогда:

```
ρ_vac(r) = ρ_vac,0·[1 - GM/(c²r)]
```

### 4.2. Уравнение для вакуумной плотности

Плотность вакуумной энергии должна удовлетворять уравнению непрерывности:

```
∇²ρ_vac + источник = 0
```

Источник — это сама масса.

**Пуассоново уравнение для ρ_vac:**

```
∇²ρ_vac = 4πG·ρ_matter·ρ_vac,0/c²
```

**Решение в сферической симметрии:**

```
ρ_vac(r) = ρ_vac,0·[1 - GM/(c²r)]
```

Проверка:

```
∇²(1/r) = -4πδ³(r)
```

```
∇²ρ_vac = ρ_vac,0·(GM/c²)·(-4πδ³(r))
```

```
∇²ρ_vac = -4πG·M·ρ_vac,0/c²·δ³(r)
```

Что совпадает с исходным уравнением при ρ_matter = M·δ³(r).

### 4.3. Эффективный потенциал

Определим **гравитационный потенциал** через вакуумную плотность:

```
Φ(r) = -c²·ln[ρ_vac(r)/ρ_vac,0]
```

Для слабых полей (GM/(c²r) ≪ 1):

```
ln[1 - x] ≈ -x
```

```
Φ(r) ≈ -c²·(-GM/(c²r)) = GM/r
```

**Ньютоновский потенциал восстановлен!**

### 4.4. Сила и ускорение

Сила на пробное тело:

```
F = -m·∇Φ = -m·GM/r²·r̂
```

Ускорение:

```
a = F/m = -GM/r²·r̂
```

**Закон всемирного тяготения получен из эффекта Казимира!**

### 4.5. Прямая связь: ускорение ← градиент энергии вакуума

**Это ключевая формула модели!**

#### **Шаг 1: Градиент плотности вакуумной энергии**

Из ρ_vac(r) = ρ_vac,0·[1 - GM/(c²r)]:

```
∇ρ_vac = -ρ_vac,0·d/dr[GM/(c²r)]·r̂

∇ρ_vac = ρ_vac,0·GM/(c²r²)·r̂
```

#### **Шаг 2: Градиент давления вакуума**

```
∇P_vac = -(c²/3)·∇ρ_vac

∇P_vac = -(c²/3)·ρ_vac,0·GM/(c²r²)·r̂

∇P_vac = -ρ_vac,0·GM/(3r²)·r̂
```

#### **Шаг 3: Сила от градиента давления**

```
F = -V_body·∇P_vac

где V_body = m/ρ — объём тела массы m

F = -(m/ρ)·[-ρ_vac,0·GM/(3r²)]·r̂

F = (m/ρ)·ρ_vac,0·GM/(3r²)·r̂
```

#### **Шаг 4: Используем связь вакуума с материей**

Из условия согласования (раздел 3.4):

```
ρ_vac,0 = 3ρ
```

Подставляем:

```
F = (m/ρ)·3ρ·GM/(3r²)·r̂

F = m·GM/r²·r̂
```

Знак минус появляется из направления градиента (к центру):

```
F = -GMm/r²·r̂
```

#### **Шаг 5: КЛЮЧЕВАЯ ФОРМУЛА — Ускорение через градиент вакуума**

```
a = F/m = (1/ρ)·ρ_vac,0·GM/(3r²)·r̂
```

Используя ρ_vac,0 = 3ρ:

```
┌──────────────────────────────────────────────┐
│                                              │
│  a = (c²/ρ_vac,0) × ∇ρ_vac                  │
│                                              │
│  или эквивалентно:                           │
│                                              │
│  a = (c²/3ρ) × ∇ρ_vac                       │
│                                              │
└──────────────────────────────────────────────┘
```

**Альтернативная форма через градиент давления:**

```
┌──────────────────────────────────────────────┐
│                                              │
│  a = -(1/ρ) × ∇P_vac                        │
│                                              │
└──────────────────────────────────────────────┘
```

#### **Физический смысл:**

**Гравитационное ускорение прямо пропорционально градиенту плотности вакуумной энергии!**

- Коэффициент пропорциональности: c²/ρ_vac,0
- Направление: от высокой плотности вакуума к низкой
- Масса M создаёт "яму" в ρ_vac → градиент → ускорение

#### **Численная проверка для Земли:**

```
∇ρ_vac(R_E) = ρ_vac,0·GM_E/(c²R_E²)
```

```
a = (c²/ρ_vac,0)·∇ρ_vac
```

```
a = (c²/ρ_vac,0)·ρ_vac,0·GM_E/(c²R_E²)
```

```
a = GM_E/R_E²
```

```
a = 6.67×10⁻¹¹ × 6×10²⁴ / (6.37×10⁶)²
```

```
a = 9.8 м/с² ✓
```

#### **Связь с эффектом Казимира:**

В классическом эффекте Казимира:

```
F_Casimir/A = -π²ℏc/(240d⁴)  (сила на площадь)
```

```
∇P_Casimir ~ ℏc/d⁵
```

В гравитации:

```
∇P_vac = -(c²/3)·∇ρ_vac

Аналогия: ∇ρ_vac играет роль "изменения числа мод"
```

**Обе силы имеют общую природу: градиент вакуумной энергии!**

## 4.6. Практическое применение: расчет гравитации через градиент вакуума

### 4.6.1. Общая формула для любой геометрии

Для **произвольного распределения масс** M(r'):

```
∇ρ_vac(r) = ∫ [ρ_vac,0·G·M(r')/(c²|r-r'|²)]·(r-r')/|r-r'| d³r'
```

**Ускорение в точке r:**

```
┌──────────────────────────────────────────────────────┐
│                                                        │
│  a(r) = (c²/ρ_vac,0) × ∫ ∇ρ_vac(r) d³r               │
│                                                        │
│       = G × ∫ [M(r')/|r-r'|²]·(r-r')/|r-r'| d³r'     │
│                                                        │
└──────────────────────────────────────────────────────┘
```

### 4.6.2. Алгоритм расчета

**Дано:** Распределение масс M(r)

**Найти:** Гравитационное ускорение a(r)

**Шаг 1:** Вычислить локальную плотность вакуума:

```
ρ_vac(r) = ρ_vac,0 × [1 - ∑ᵢ GMᵢ/(c²|r-rᵢ|)]
```

**Шаг 2:** Вычислить градиент:

```
∇ρ_vac(r) = ρ_vac,0 × ∑ᵢ [GMᵢ/(c²|r-rᵢ|²)]·r̂ᵢ
```

где r̂ᵢ = (r-rᵢ)/|r-rᵢ|

**Шаг 3:** Вычислить ускорение:

```
a(r) = (c²/ρ_vac,0) × ∇ρ_vac(r)
```

**Упрощение:**

```
a(r) = ∑ᵢ [GMᵢ/|r-rᵢ|²]·r̂ᵢ
```

**Это стандартная формула Ньютона, выведенная из градиента вакуума!**

### 4.6.3. Пример 1: Однородная сфера

**Дано:**
- Масса M
- Радиус R
- Точка на расстоянии r > R

**Решение:**

```
∇ρ_vac(r) = ρ_vac,0·GM/(c²r²)·r̂

a(r) = (c²/ρ_vac,0)·ρ_vac,0·GM/(c²r²)·r̂

a(r) = GM/r²·r̂  (направлено к центру)

|a| = GM/r²
```

**Для Земли на поверхности:**

```
|a| = 6.67×10⁻¹¹ × 6×10²⁴ / (6.37×10⁶)²
|a| = 9.8 м/с²  ✓
```

### 4.6.4. Пример 2: Две массы (система Земля-Луна)

**Дано:**
- M_E (Земля), M_L (Луна)
- Расстояние r_EL = 384,400 км
- Точка между ними на расстоянии x от Земли

**Решение:**

```
∇ρ_vac(x) = ρ_vac,0/(c²) × [GM_E/x² - GM_L/(r_EL-x)²]·x̂
```

```
a(x) = GM_E/x² - GM_L/(r_EL-x)²
```

**Точка Лагранжа L1** (где a = 0):

```
GM_E/x² = GM_L/(r_EL-x)²
```
Решая:
```
x_L1 ≈ 326,000 км от Земли  ✓
```

### 4.6.5. Пример 3: Гравитация внутри планеты

**Дано:**
- Однородная сфера массы M, радиуса R
- Точка внутри на расстоянии r < R

**Решение:**

Только масса внутри сферы радиуса r создаёт градиент:

```
M_внутри = M × (r/R)³
```

```
∇ρ_vac(r) = ρ_vac,0·G·M_внутри/(c²r²)
```

```
a(r) = G·M·r/R³
```

```
|a| ∝ r  (линейно растёт с расстоянием от центра)
```

**На поверхности (r = R):**

```
a(R) = GM/R²  ✓
```

**В центре (r = 0):**

```
a(0) = 0  ✓
```

### 4.6.6. Пример 4: Тонкий диск

**Дано:**
- Диск массы M, радиуса R
- Точка на оси на расстоянии z от центра

**Решение:**

Интегрируем по кольцам:

```
∇ρ_vac(z) = ρ_vac,0·GM/(c²) × ∫₀ᴿ [z/(r²+z²)^(3/2)] × (2πr dr)/(πR²)
```

```
a(z) = (2GM/R²) × [1 - z/√(R²+z²)]
```

Предел z → 0:
```
a(0) = 2GM/R²  (в 2 раза больше, чем для точечной массы!)
```

Предел z → ∞:
```
a(z) → GM/z²  (как точечная масса)
```

### 4.6.7. Пример 5: Бесконечная плоскость

**Дано:**
- Поверхностная плотность σ (кг/м²)
- Точка на расстоянии z от плоскости

**Решение:**

```
∇ρ_vac(z) = ρ_vac,0 × 2πGσ/(c²) × sign(z)
```

```
a(z) = 2πGσ × sign(z)
```

```
|a| = const  (не зависит от расстояния!)
```

**Численно для σ = 10⁴ кг/м²:**

```
|a| = 2π × 6.67×10⁻¹¹ × 10⁴ = 4.2×10⁻⁶ м/с²
```

## 5. Вывод закона всемирного тяготения

### 5.1. Система двух масс

Рассмотрим две массы M₁ и M₂ на расстоянии r.

**Каждая масса модифицирует вакуум:**

Масса M₁:
```
ρ_vac,1(r) = ρ_vac,0·[1 - GM₁/(c²r)]
```

Масса M₂:
```
ρ_vac,2(r) = ρ_vac,0·[1 - GM₂/(c²r)]
```

**Суперпозиция модификаций:**

В точке между массами:

```
ρ_vac,total = ρ_vac,0·[1 - GM₁/(c²r₁) - GM₂/(c²r₂)]
```

### 5.2. Градиент давления

Давление вакуума:


## 6. Вывод уравнений Эйнштейна

### 6.1. Метрика в терминах вакуумной плотности

Плотность вакуумной энергии определяет метрику пространства-времени:

```

```

```

```

**Шварцшильдовская метрика в слабом приближении!**

### 6.2. Тензор энергии-импульса вакуума

Вакуум имеет тензор энергии-импульса:

```
T^μν_vac = ρ_vac·diag(1, -1, -1, -1)
```

(плотность энергии + отрицательное давление по всем направлениям)

### 6.3. Уравнения Эйнштейна

Уравнения Эйнштейна:

```
G_μν = 8πG/c⁴·T_μν
```

где G_μν — тензор Эйнштейна.

**Левая часть** (из метрики):

```

```

(уравнение Пуассона для слабого поля)

**Правая часть** (из вакуумного тензора):

Полный тензор энергии-импульса:

```
T_μν = T_μν,matter + T_μν,vac
```

В нашей модели вакуум **реагирует** на материю:

```
ρ_vac(r) = ρ_vac,0·[1 - GM/(c²r)]
```

Это эквивалентно эффективному тензору:

```
T^eff_μν = T_μν,matter + [модификация от градиента ρ_vac]
```

**Подставляя в уравнения Эйнштейна:**

```
∇²Φ = 4πG·ρ_matter + поправка от ∇²ρ_vac
```

В линейном приближении поправка мала → **стандартные уравнения Эйнштейна!**

**Точная форма:**
```
z_exact = [1 - GM/(c²r)]⁻¹/² - 1
```

#### **Формула 6: Горизонт событий через вакуум**

**Классическая:**
```
r_Schwarzschild = 2GM/c²
```

**Новая (из модели Казимира):**

Горизонт = поверхность, где ρ_vac → 0:

```
ρ_vac(r_horizon) = 0
```

```
1 - GM/(c²r_horizon) = 0
```

```
r_horizon = GM/c²
```

**Радиус Шварцшильда:**
```
r_Sch = 2 × r_horizon = 2GM/c²  ✓
```

**Физический смысл:** На горизонте вакуумная энергия **обращается в ноль** → критическая модификация пространства-времени.

#### **Формула 7: Сила между двумя массами (новое выражение)**

**Классическая:**
```
F₁₂ = G M₁M₂/r²
```

**Новая (из модели Казимира):**
```
F₁₂ = -(V₁ × V₂) × ∇₁∇₂[ρ_vac,0 × c²/3]
```

```
    = (M₁/ρ₁) × (M₂/ρ₂) × (ρ₁ρ₂) × G/r²
```

```
    = G M₁M₂/r²  ✓
```

где используется **связь плотностей:**
```
ρ_vac,0 = 3ρ_matter
```

### 11.6. Визуальная схема вычисления


┌─────────────────────────────────────────────────────────────┐
│         КАК ВЫЧИСЛИТЬ ГРАВИТАЦИОННОЕ УСКОРЕНИЕ              │
└─────────────────────────────────────────────────────────────┘

МЕТОД 1: Классический (Ньютон)
──────────────────────────────
Масса M → Сила «F = GMm/r²» → «a = F/m = GM/r²»


МЕТОД 2: Через потенциал
────────────────────────
Масса M → «Φ(r) = -GM/r» → «a = -∇Φ = GM/r²»


МЕТОД 3: Через вакуум (НОВЫЙ!)
───────────────────────────────
Масса M → «ρ_vac(r) = ρ_vac,0[1 - GM/(c²r)]»
        ↓
   «∇ρ_vac(r) = ρ_vac,0·GM/(c²r²)·r̂»
        ↓
   «a = (c²/ρ_vac,0)·∇ρ_vac = GM/r²»

═══════════════════════════════════════

Классика:    Масса притягивает массу (магия?)
ОТО:         Масса искривляет пространство
Казимир:     Масса модифицирует вакуум → градиент → сила


### 11.7. Практическое применение: калькулятор гравитации

#### **Задача:** Найти ускорение в точке r от массы M

**ВХОД:**
- M = масса источника (кг)
- r = расстояние (м)
- ρ_vac,0 = 3ρ_matter ≈ 10⁻²⁶ кг/м³

**Алгоритм:**

```python
# Шаг 1: Вычислить модификацию вакуума
rho_vac_r = rho_vac_0 * (1 - G*M/(c**2 * r))

# Шаг 2: Вычислить градиент
grad_rho_vac = rho_vac_0 * G*M / (c**2 * r**2)  # модуль

# Шаг 3: Вычислить ускорение
a = (c**2 / rho_vac_0) * grad_rho_vac
a = G*M / r**2  # упрощается!

print(f"Ускорение: {a:.2f} м/с²")
```

**ПРИМЕР: Земля**

```python
M = 6e24  # кг
r = 6.37e6  # м (радиус Земли)
G = 6.67e-11
c = 3e8
rho_vac_0 = 1e-26  # кг/м³

# Прямой расчёт
a_newton = G * M / r**2
# a_newton = 9.8 м/с²

# Через вакуум
grad_rho = rho_vac_0 * G*M / (c**2 * r**2)
# grad_rho = 4.9e-55 кг/м⁴

a_vacuum = (c**2 / rho_vac_0) * grad_rho
# a_vacuum = 9.8 м/с²  ✓ СОВПАДАЕТ!
```

### 11.8. Ключевые новшества модели

#### **1. Единая формула для всех режимов**

```
ρ_vac(r) = ρ_vac,0 × [1 - GM/(c²r)]
```

Эта **одна формула** даёт:
- Слабые поля (r → ∞): Ньютоновская гравитация
- Сильные поля (r ~ r_Sch): Эффекты ОТО
- Горизонт (r = r_Sch/2): ρ_vac → 0

#### **2. Физический механизм G**

```
G = ℏc⁵/(ρ_vac,0 × l⁴_cutoff)
```

G **не фундаментальна**, а определяется:
- Плотностью вакуумных флуктуаций ρ_vac,0
- Масштабом обрезания l_cutoff ~ мм

#### **3. Динамическая космологическая постоянная**

```
Λ(t) = 8πG × ρ_vac,0(t)/c²
```

Λ **изменяется** с эволюцией Вселенной!

#### **4. Квантовые поправки естественны**

Флуктуации ρ_vac → флуктуации метрики:

```
⟨δg_μν⟩ ~ δρ_vac/ρ_vac,0 ~ (l_Planck/L)⁴
```

### 11.6. Практическое применение формул

#### **Задача 1: Орбитальная скорость**

**Классически:**
```
v = √(GM/r)
```

**Из модели Казимира:**
Центростремительная сила = Казимирова сила

```
mv²/r = m × |∇P_vac| × V_body
```

```
v² = (V_body/m) × ρ_vac,0 × GM/(3c²r²)
```

используя:

```
V_body/m = 1/ρ и ρ_vac,0 = 3ρ
```

```
v² = GM/r  ✓
```

#### **Задача 2: Период обращения**

```
T = 2πr/v = 2π√(r³/GM)
```

Через вакуум:

```
T = 2π√[r³c²/(ρ_vac,0 × r²/3)]
```

```
  = 2π√(3r³c²/ρ_vac,0)
```

```
  = 2π√(r³/GM)  ✓
```

#### **Задача 3: Первая космическая скорость**

На поверхности планеты массы M, радиуса R:

```
v₁ = √(2GM/R)
```

**Из модели Казимира:**

Плотность вакуума на поверхности планеты (расстояние R):

```
ρ_вакуума(R) = ρ_вакуума,свободное × [1 - (GM)/(c²R)]
```

где:
  ρ_вакуума,свободное — плотность вакуума вдали от масс
  G — гравитационная постоянная
  M — масса планеты
  c — скорость света
  R — радиус планеты

**Энергия для выхода** = изменение вакуумной энергии:

```
ΔE = ∫ ∇P_вакуума · dr = (GMm)/R
```

**Кинетическая энергия** = потенциальная энергия:

```
(m × v₁²)/2 = (GMm)/R
```

```
v₁² = 2GM/R
```

```
v₁ = √(2GM/R)  ✓
```

## Заключение

**Гравитация есть обобщённый эффект Казимира.**

Массы изменяют граничные условия для квантовых полей вакуума. Градиент вакуумной энергии создаёт наблюдаемую гравитационную силу. Все классические и релятивистские эффекты гравитации выводятся из этого принципа.

Модель предоставляет **физически ясную картину** гравитации как **макроскопического квантового эффекта**, объединяя гравитацию с другими взаимодействиями через общую вакуумную структуру.

**Ключевые формулы модели:**

```
┌─────────────────────────────────────────────────────────┐
│                   БАЗОВАЯ ФОРМУЛА                         │
│                                                           │
│  ρ_vac(r) = ρ_vac,0 × [1 - GM/(c²r)]                    │
│                                                           │
└─────────────────────────────────────────────────────────┘
        ↓ градиент
┌─────────────────────────────────────────────────────────┐
│              КЛЮЧЕВАЯ ФОРМУЛА УСКОРЕНИЯ                   │
│                                                           │
│  a = (c²/ρ_vac,0) × ∇ρ_vac                              │
│                                                           │
│  ГРАВИТАЦИОННОЕ УСКОРЕНИЕ = ГРАДИЕНТ ВАКУУМНОЙ ЭНЕРГИИ   │
│                                                           │
└─────────────────────────────────────────────────────────┘
```

Из этих **двух формул** следуют:
- Закон всемирного тяготения Ньютона ✓
- Уравнения Эйнштейна ✓
- Красное смещение ✓
- Чёрные дыры ✓
- Космологическая постоянная ✓

**Гравитация — это не искривление пространства, а градиент вакуумной энергии!**
