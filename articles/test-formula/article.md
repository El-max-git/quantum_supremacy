# Тестовая статья с проблемной формулой

Это тестовая статья для проверки отображения проблемной формулы из статьи про вселенную.

## Связь со скоростью роста гиперплощади

Поскольку $\text{div}(g) = 6H$ и $\dot{V}/V = 3H$:

$$\text{div}(g) = 2 \times \left(\frac{\dot{V}}{V}\right)$$

Эволюция дивергенции **напрямую определяет** эволюцию скорости роста гиперплощади.

## Другие формулы для сравнения

Простая inline формула: $E = mc^2$

Простая block формула:

$$H(t) = \frac{\dot{R}(t)}{R(t)}$$

Формула с \text{}:

$$\text{div}(g) = 6H(t)$$

Формула с \left(\right):

$$\frac{\dot{V}}{V} = 3H(t)$$

## 6.3. Энергия Казимира на космологических масштабах

**Эффект Казимира между пластинами:**

$$\rho_{\text{Casimir}} \sim \frac{\hbar c}{d^4}$$

**Для космологических расстояний d ~ R:**

$$\rho_{\text{Casimir}} \sim \frac{\hbar c}{R^4}$$

При $R \sim 10^{26} \text{ м}$:

$$\rho_{\text{Casimir}} \sim \frac{10^{-34}}{(10^{26})^4} \sim 10^{-138} \text{ Дж/м}^3 \ll \rho_{\text{vac}}$$

**Вывод:** простой эффект Казимира на космологических масштабах ничтожен.

**Но!** Если учесть **все моды** с разными длинами волн, вклад может быть значительнее (проблема регуляризации).
